# Control-de-LEDs-y-Pulsador-con-ATmega328P
Este proyecto implementa el control de periféricos externos utilizando los puertos de I/O de un ATmega328P. El sistema controla 8 LEDs conectados al puerto B, permitiendo encenderlos mediante distintas **secuencias de iluminación**, las cuales pueden alternarse utilizando un **pulsador**.  Además, se implementa un algoritmo antirebote por software
